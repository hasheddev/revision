GREP
grep options  term   filepath OR cat file | grep options searchterm
grep '^term' filepath    single quotes prevents bash from interpreting special characters
grep -A1 term file.txt  print term and one line after it (number = how many lines to print)
grep -B1 term file.txt  print term and one line before it
grep -C1 term file.txt  print term and one line before and after it
grep -i term file.txt  find case insensitively the given term
grep -o te file.txt   print only exact text that match (would print only te)
grep -i name file.txt | grep newterm

?pattern	Search backward for a specific term
